# 🏀 StatWars

StatWars is an NBA-focused social competitive platform where users play interactive mini-games, challenge friends, and test their basketball knowledge through real-time and turn-based gameplay.

---

## 🚀 Features

- 🎮 **Card Battle Game**
  - Players receive NBA player cards
  - Each round presents a stat (points, rebounds, assists)
  - Highest stat wins the round

- 🧠 **Ranking Game**
  - Users rank players based on a given statistic
  - Points awarded based on accuracy

- ❓ **Trivia / Guess Game**
  - Answer NBA-related questions
  - Test your basketball knowledge

- 🏆 **Weekly Fantasy Quiz (Planned)**
  - Compete in weekly tournaments
  - Leaderboard-based ranking system

- 🧑‍🤝‍🧑 **Multiplayer Gameplay**
  - Play with friends
  - Real-time and turn-based modes

---

## 🎯 Vision

StatWars aims to become a **social NBA knowledge platform** where users:
- Compete with others
- Improve their basketball knowledge
- Enjoy multiple engaging mini-games

---

## 🛠️ Tech Stack

- **Frontend:** React (Next.js)
- **Backend:** Node.js (Express / NestJS)
- **Realtime:** Socket.io (WebSockets)
- **Database:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth / JWT
- **Hosting:** Vercel & Render
